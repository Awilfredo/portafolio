Seleccion de proyectos

Mi enfoque se centra en la creación de soluciones elegantes y efectivas que brinden una excelente experiencia de usuario. Estos son algunos proyectos realizados.

Menú en linea

Mi proyecto consiste en crear un menú en línea para un restaurante con el objetivo de facilitar el proceso de pedido para los clientes y mejorar la eficiencia de los servicios del restaurante. La plataforma proporcionará una interfaz intuitiva y fácil de usar que permitirá a los clientes navegar a través de los diferentes menús y ver los platos que deseen pedir. Los menús se actualizarán automáticamente en tiempo real para reflejar los cambios y los nuevos platos ofrecidos por el restaurante.

Tecnologías utilizadas: Para desarrollar la plataforma de menú en línea, usé tecnologías modernas de frontend JavaScript, HTML, CSS y Boostrap, así como tecnologías de backend como PHP y MYSQL para el procesamiento de datos.

Encriptador de texto

Mi proyecto consiste en modificar las vocales de un texto ingresado para generar un texto encriptado.

Tecnologías utilizadas:
Para desarrollar la plataforma del encriptador de texto, usé las tecnologías de JavaScript, HTML y CSS.

Juego ahorcado

Este proyecto es un juego de palabras en el que el usuario ingresa una letra y si no es correcta se dibuja la silueta de una persona. El usuario solo tiene seis intentos para poder ganar.

Tecnologías usadas:
para desarrollar la plataforma del juego ahorcado, hice uso las tecnologías de Canvas, JavaScript, HTML y CSS.
